
# react-practice

## Environment
node.js v14.21.3

npm v6.14.18

## react example
```
$ npx create-react-app REACT_APP_NAME
$ cd REACT_APP_NAME
$ npm start
```

## react start
```
$ npm update
$ npm start
```

## Ant Design
[Ant Design](https://ant.design/)
```
$ npm install antd
```

## SCSS
```
$ npm install node-sass
```

## swiper
